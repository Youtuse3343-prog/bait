Render settings:

Service type: Web Service (because this bot also runs a dashboard)
Build command: pip install -r requirements.txt
Start command: python bot.py

Important environment variables:
BOT_TOKEN=your bot token
MONGO_URI=your MongoDB URI
DISCORD_CLIENT_ID=your Discord application client ID
DISCORD_CLIENT_SECRET=your Discord OAuth secret
PUBLIC_BASE_URL=https://your-render-service.onrender.com
OWNER_USER_ID=your Discord user ID
SYNC_COMMANDS=false
ENABLE_SELF_PING=false

When you edit/add slash commands, temporarily set SYNC_COMMANDS=true for one successful deploy, then set it back to false.
Do not run this same bot token locally while Render is running.
